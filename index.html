AI-ANK

Private On-Device AI for Android

What is AI-ANK?

AI-ANK is a private AI assistant that runs entirely on your Android device. No data leaves your phone, no cloud, no subscription. All conversations happen locally using Google's Gemma language model.

Getting Started

First Launch

The first time you open AI-ANK, it automatically downloads the AI model to your device. This only happens once.

⬇️

Model download (~2.6 GB)

On first launch, AI-ANK downloads Gemma-4-E2B, Google's 2-billion parameter language model, directly to your device. The download runs in the background — you'll see a notification with live progress. You don't need to keep the app open or the screen on.

Download time: roughly 5–15 minutes on a good Wi-Fi connection. Recommended to use Wi-Fi rather than mobile data.
💬

Start chatting

Once the model finishes downloading, the chat window opens automatically. From this point on, no internet connection is needed for conversations — everything runs on your device.

Core Feature

Chatting

Type your message in the bar at the bottom and tap Send. The model processes everything on-device.

✅

What it's good at

Writing, summarising, explaining concepts, answering general knowledge questions, brainstorming, coding help, drafting messages and emails.

⚠️

Expectations for a 2B model

This is a compact model that runs on a phone — impressively capable for its size, but not GPT-4. Complex multi-step reasoning or very long documents may produce mixed results.

📎

File & image attachments

Tap the + icon in the chat bar to attach images, PDFs, or documents. The model can read and discuss the content.

Feature

Web Search

AI-ANK can search the web for current information — useful for questions the offline model wouldn't know.

1

Enable web search

Tap the Search button in the chat bar (bottom left). It highlights when active. You need an internet connection for this feature.

2

Ask a current question

Examples that work well:

  • "What was the latest release by OpenAI?"
  • "Current price of gold"
  • "Latest news on [topic]"
Note: Web search works on simple factual queries. It uses search snippets rather than full article content, so answers to very recent or complex news may be incomplete. This will be improved in the next release.

Models

Choosing a Model

Tap the model name in the top bar → Models to download and switch between models.

⚡

Gemma-4-E2B Default

Google's 2-billion parameter model. Fast responses, smaller download (~2.6 GB). Good for everyday tasks — writing, Q&A, summarising. Recommended starting point.

🧠

Gemma-4-E4B

4-billion parameter model. Noticeably better quality on complex questions and longer conversations. Larger download (~3.6 GB) and slightly slower on lower-end devices. Recommended for flagship phones with 8 GB+ RAM.

🔜

More models coming Soon

Larger models with GPT-4-level capability, plus dedicated audio and video models, are on the roadmap.

Storage tip: You can download multiple models and switch between them any time. Delete a model from the Models screen to free up space.

Reference

Settings

Tap the ⚙️ icon (top right) to open Settings. Most users won't need to change these.

System Instruction
A custom prompt that shapes the model's personality and style. You can rewrite this to give the model a different persona or focus area.
Backend
GPU (default) — fastest on most phones. CPU — slower but more compatible. NPU — uses the phone's neural processor if supported. Stick with GPU unless you're having issues.
Temperature
Controls creativity. Lower (e.g. 0.3) = more predictable, factual answers. Higher (e.g. 1.0) = more creative and varied. Default is 0.8.
Max Tokens
Maximum length of each response. Higher = longer answers but slower generation. Default is 2048.
Top K / Top P
Advanced sampling controls. Leave at defaults unless you know what you're doing.
Theme
Light, Dark, or follow System setting.
