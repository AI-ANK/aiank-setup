AI-ANK Setup Guide

Get up and running in a few minutes

What you'll need

AI-ANK runs AI models entirely on your device — nothing leaves your phone. To download the models, you need a free HuggingFace account and an access token. This guide walks you through it.

Step-by-step

1

Create a HuggingFace account

Go to huggingface.co/join and sign up for a free account. No credit card required.

2

Accept the Gemma model license Required

The Gemma models used by AI-ANK are gated — you must accept Google's license before you can download them.

  • Visit Gemma-4-E2B (2B model)
  • Visit Gemma-4-E4B (4B model)
  • Click Agree and access repository on each page
Note: You only need to do this once per model. Your acceptance is tied to your HuggingFace account.
3

Generate an access token

Go to huggingface.co/settings/tokens and create a new token:

  • Click New token
  • Give it any name (e.g. AI-ANK)
  • Set role to Read
  • Click Generate a token
  • Copy the token — it starts with hf_
4

Paste the token into AI-ANK

In the app, open Settings (gear icon, top right) → scroll to HuggingFace → paste your token into the Access Token field.

The token is stored only on your device and is never sent anywhere except HuggingFace for model downloads.
5

Download a model

Tap the model name in the top bar → open Model Manager → select a model and tap Download. The download runs in the background.

  • Gemma-4-E2B — ~1.8 GB, faster, good for most tasks
  • Gemma-4-E4B — ~3.2 GB, more capable

✅ You're all set!

Once the model finishes downloading, start a chat. Everything runs on your device — no internet needed for conversations.
